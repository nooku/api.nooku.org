public string
# getFormat( )

Forces format to "rss" if it comes in as "feed" per Joomla conventions if SEF suffixes are enabled

Forces format to "rss" if it comes in as "feed" per Joomla conventions if SEF suffixes are enabled

Find the format by using following sequence :

  1. Use the the 'format' request parameter
  2. Use the URL path extension
  3. Use the accept header with the highest quality apply the reverse format map to find the format.

string
The request format or NULL if no format could be found

# getPort( )

If PHP is on a secure connection always return 443 instead of 80

If PHP is on a secure connection always return 443 instead of 80

When PHP is behind a reverse proxy port information might not be forwarded correctly. Also, $_SERVER['SERVER_PORT'] is not configured correctly on some hosts and always returns 80.

This method can read the client port from the "X-Forwarded-Port" header when the request is proxied and the proxy is trusted. The "X-Forwarded-Port" header must contain the client port.
