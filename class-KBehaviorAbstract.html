Class KBehaviorAbstract

Abstract Behavior

The abstract behavior will translate the command name to a method name format (eg, _before[Command] or _after[Command]) and add execute the method. Command handlers should be declared protected.

Get the methods that are available for mixin based

Function will check if the behavior is supported by calling KBehaviorAbstract::isSupported(). Is the behavior is not supported on the mixer no mixable methods will be returned, only an 'is[Behavior]' method will be added which return FALSE when called.
