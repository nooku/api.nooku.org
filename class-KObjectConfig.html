Class KObjectConfig

KObjectConfig

KObjectConfig provides a property based interface to an array. Data is can be modified unless the object is marked as readonly.

KObjectConfig implements KObjectConfigInterface

Direct known subclasses

KCommand, KControllerToolbarCommand, KEvent, KModelPaginator, KObjectConfigFormat

Indirect known subclasses

KControllerContext, KDatabaseContext, KViewContext, KViewContextTemplate, KDispatcherContext, KEventException, KModelContext, KObjectConfigIni, KObjectConfigJson, KObjectConfigPhp, KObjectConfigXml, KObjectConfigYaml

# merge( array|Traversable|KObjectConfig $options )

Merge options

Merge options

This method will overwrite keys that already exist, keys that don't exist yet will be added.

For duplicate keys, the following will be performed:

  • Nested configs will be recursively merged.
  • Items in $options with INTEGER keys will be appended.
  • Items in $options with STRING keys will overwrite current values.
