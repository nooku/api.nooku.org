Class KUser

User Singleton

User is the user implementation used by the in-memory user provider. This object is tightly coupled to the session. all data is stored and retrieved from the session attribute container, using a special 'user' namespace to avoid conflicts.

KObject implements KObjectInterface, KObjectMixable, KObjectHandlable, KObjectDecoratable
KUserAbstract implements KUserInterface
KUser implements KObjectSingleton
