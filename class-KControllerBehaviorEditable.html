# _actionSave( KControllerContextInterface $context )

Save action

Save action

This function wraps around the edit or add action. If the model state is unique a edit action will be executed, if not unique an add action will be executed.

This function also sets the redirect to the referrer if the action succeeds and will redirect to the current url if the edit/add action fails while setting the status message.
