Class KObjectSet

Object Set

A set is a data structure that can store objects, without any particular order, and no repeated values. Unlike most other collection types, rather than retrieving a specific element from a set, one typically tests if a object is contained in the set.

KObjectSet implements an associative container that stores objects, and in which the object themselves are the keys. Objects are stored in the set in FIFO order.

KObject implements KObjectInterface, KObjectMixable, KObjectHandlable, KObjectDecoratable
KObjectSet implements IteratorAggregate, ArrayAccess, Countable, Serializable
