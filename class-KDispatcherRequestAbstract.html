# getAddress( )

Returns the client IP address.

Returns the client IP address.

This method can read the client port from the "X-Forwarded-For" header when the request is proxied and the proxy is trusted. The "X-Forwarded-For" header must contain the client address. The "X-Forwarded-For" header value is a comma+space separated list of IP addresses, the left-most being the original client, and each successive proxy that passed the request adding the IP address where it received the request from.

Returns

string
Client IP address or an empty string if it's not supplied in the request

Link

http://tools.ietf.org/html/draft-ietf-appsawg-http-forwarded-10#section-5.2
